Ce projet consiste à mettre en application les enseignements du cours "Gérez votre code avec Git et GitHub" du site openclassrooms

L'activité consiste àcreer un repository Git qui détient :

- un fichier README.txt : description du projet
- deux fichiers quelconque
- et effectuer quatre commits avec des modifications

Ce repository sera envoyé sur GitHub afin de le rendre visible à tous.
sont URL aura la forme suivante : https://github.com/marubric/Activite
